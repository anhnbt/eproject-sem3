# Flowcharts - Sơ đồ luồng các quy trình chính

## 1. Quy trình quản lý người dùng (User Management)

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|    Bắt đầu     |---->| Nhập thông tin |---->| Kiểm tra thông |
|                |     | người dùng     |     | tin đã nhập    |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
                                               |
                                               |
                                               v
+----------------+                       +----------------+
|                |                       |                |
| Lưu thông tin  |<---------------------<  Thông tin hợp  |
| vào CSDL       |          Có           |  lệ?           |
|                |                       |                |
+----------------+                       +----------------+
      |                                        |
      |                                      Không
      v                                        |
+----------------+     +----------------+      |
|                |     |                |<-----+
| Gửi xác nhận   |---->|  Kết thúc      |
| đến người dùng |     |                |
|                |     |                |
+----------------+     +----------------+
```
