# Flowcharts - Sơ đồ luồng các quy trình chính

## 1. Quy trình đăng ký bệnh nhân mới

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|    Bắt đầu     |---->| Nhập thông tin |---->| Kiểm tra thông |
|                |     | bệnh nhân      |     | tin đã nhập    |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
                                               |
                                               |
                                               v
+----------------+                       +----------------+
|                |                       |                |
| Lưu thông tin  |<---------------------<  Thông tin hợp  |
| vào CSDL       |          Có           |  lệ?           |
|                |                       |                |
+----------------+                       +----------------+
      |                                        |
      |                                      Không
      v                                        |
+----------------+     +----------------+      |
|                |     |                |<-----+
| Thông báo đăng |---->|  Kết thúc      |
| ký thành công  |     |                |
|                |     |                |
+----------------+     +----------------+
```

## 2. Quy trình đặt lịch hẹn

```
+----------------+     +----------------+     +----------------+

|    Bắt đầu     |---->| Chọn bác sĩ và |---->| Kiểm tra lịch  |
|                |     | ngày giờ       |     | làm việc bác sĩ|
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
                                               |
                                               |
                                               v
+----------------+                       +----------------+
|                |                       |                |
| Tạo lịch hẹn   |<---------------------<  Bác sĩ có sẵn |
| mới            |          Có           |  sàng?         |
|                |                       |                |
+----------------+                       +----------------+
      |                                        |
      |                                      Không
      v                                        |
+----------------+     +----------------+      |
|                |     |                |<-----+

| xác nhận       |     |                |
|                |     |                |
+----------------+     +----------------+
```

## 3. Quy trình khám bệnh

```
+----------------+     +----------------+     +----------------+
                                               |
                                               |
                                               v

|                |     |                |     |                |
| Ghi nhận kết   |<----| Bác sĩ khám    |<----| Hướng dẫn vào  |
| quả khám       |     | bệnh           |     | phòng khám     |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
      |
      |
      v
+----------------+                       +----------------+
|                |                       |                |
| Tạo đơn thuốc  |---------------------->| Chuyển bệnh    |
| (nếu cần)      |                       | nhân đến quầy  |

                                               |
                                               |
                                               v
                                        +----------------+
                                        |                |
                                        |  Kết thúc      |
                                        |                |
                                        |                |
                                        +----------------+
```
